# 02. 웹 리서치 지침

## 목적

입력된 상품정보를 기반으로 유사 상품의 상세페이지 구성, 카피 패턴, 구매 설득 요소, 이미지 연출 방식을 조사한다.

## 검색 키워드 생성 규칙

```text
[상품명] 상세페이지
[카테고리] 상세페이지
[카테고리] 추천
[카테고리] 스마트스토어
[카테고리] 쿠팡 인기
[카테고리] 장점
[카테고리] 구매포인트
[핵심 장점] [카테고리]
```

## 수집 항목

| 항목 | 목적 |
|---|---|
| 경쟁 상세페이지 구조 | 섹션 흐름 참고 |
| 메인 카피 패턴 | 설득 방식 분석 |
| 핵심 장점 표현 | 고객 반응 포인트 파악 |
| 이미지 연출 방식 | 섹션별 이미지 방향 설정 |
| 리뷰/불만 키워드 | 고객 니즈 분석 |
| 가격대/포지셔닝 | 가성비/프리미엄 방향 결정 |
| 스펙 표기 방식 | 정보 정리 방식 참고 |

## 주의사항

경쟁사 문구는 그대로 사용하지 않는다.  
구조와 패턴만 분석하고, 최종 카피는 새로 작성한다.

## 출력 형식

```markdown

## 검색 키워드
- 

## 참고한 상품/페이지 유형
- 

## 공통 상세페이지 구조
1. 
2. 
3. 

## 자주 보이는 카피 패턴
- 

## 고객 니즈 / 문제점
- 

## 이미지 연출 패턴
- 

## 차별화 방향
- 

## 기획 반영 포인트
- 
```

## 완료 기준

- 공통 섹션 구조 정리
- 구매 포인트 정리
- 카피 패턴 정리
- 고객 니즈 정리
- 이미지 방향 정리
