# 03. 상세페이지 기획안 작성 지침

## 목적

상품정보와 웹 리서치 결과를 바탕으로 상세페이지 전체 전략과 섹션 구성을 확정한다.

## 기획 필수 항목

| 항목 | 설명 |
|---|---|
| positioning | 상품 포지셔닝 |
| main_concept | 상세페이지 메인 콘셉트 |
| main_message | 핵심 메시지 |
| main_copy | 메인 카피 |
| sub_copy | 서브 카피 |
| target_customer | 타깃 고객 |
| conversion_strategy | 구매 전환 전략 |
| differentiation_points | 차별화 포인트 |
| section_flow | 섹션 흐름 |

## 섹션 기본 구조

| 순서 | 섹션 ID | 섹션명 | 목적 |
|---|---|---|---|
| 1 | section_01_hero | Hero | 첫인상과 메인 카피 |
| 2 | section_02_problem | Problem | 고객 문제 제기 |
| 3 | section_03_benefit | Benefit | 핵심 장점 |
| 4 | section_04_usage | Usage | 사용 장면 |
| 5 | section_05_detail | Detail | 디테일/원물/소재 |
| 6 | section_06_spec | Spec | 상품 정보 |
| 7 | section_07_guide | Guide | 사용법/보관법 |
| 8 | section_08_closing | Closing | 구매 설득 마무리 |

## 출력 형식

```markdown
# 상세페이지 기획안

## 1. 상품 포지셔닝

## 2. 메인 콘셉트

## 3. 핵심 메시지

## 4. 메인 카피

## 5. 서브 카피

## 6. 타깃 고객

## 7. 구매 설득 포인트

## 8. 차별화 포인트

## 9. 상세페이지 섹션 구성표

| 순서 | 섹션 | 목적 | 핵심 문구 | 이미지 방향 |
|---|---|---|---|---|
```

## 완료 기준

- 메인 콘셉트 확정
- 메인 카피 확정
- 섹션 8개 구성 확정
- 섹션별 목적과 이미지 방향 확정
