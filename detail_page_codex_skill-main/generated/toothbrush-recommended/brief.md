# 칫솔 상세페이지 이미지 기획안

## 1. 카테고리 및 판매 맥락 요약

| 항목 | 내용 |
|---|---|
| 카테고리 | 생활용품 / 구강케어 |
| 판매 채널 | 채널 무관 모바일 상세페이지 |
| 주요 고객 | 매일 쓰는 칫솔을 합리적으로 고르는 가족/1인가구 고객, 잇몸 부담과 세정력을 함께 보는 고객 |
| 핵심 구매 이유 | 칫솔모 밀도, 헤드 크기, 그립감, 위생 교체 주기를 한눈에 이해 |
| 예상 구매 불안 | 칫솔모가 너무 딱딱하지 않은지, 치아 사이 세정이 되는지, 헤드가 큰지, 구성 수량이 맞는지 |
| 권장 컷 수 | 12컷 기본형 |
| 상품 사진 | 없음: 판매용 초안 |
| 상세페이지 스타일 | 쿠팡 실용 정보형 추천 |

## 2. 상세페이지 핵심 전략

칫솔은 매일 쓰는 저관여 생활용품이라 첫 화면에서 “왜 이 칫솔을 골라야 하는지”가 빠르게 보여야 한다. 추천 구조는 세정력, 잇몸 부담, 그립감, 구성/교체 주기 순서로 구매 불안을 줄이는 쿠팡 실용 정보형이다.

사진이 없으므로 최종 이미지는 일반적인 칫솔 비주얼 기반의 `판매용 초안`으로 제작해야 한다. 브랜드명, 실제 색상, 칫솔모 강도, 구성 수량, 인증/소재 정보는 확인 전까지 확정 주장으로 넣지 않는다.

## 3. 이미지 컷별 제작안

## 컷 1. 메인 히어로

**목적:** 칫솔의 핵심 구매 이유를 첫 화면에서 전달한다.

**헤드라인:** 매일 쓰는 칫솔, 더 꼼꼼하게 고르세요

**서브카피:** 칫솔모, 헤드, 그립감까지 한눈에 확인하는 데일리 구강케어 칫솔.

**이미지 내 삽입 문구:** `데일리 칫솔` / `칫솔모 · 헤드 · 그립감 확인` / `구성 정보 확인 필요`

**이미지 구성:** 흰 배경에 칫솔 3~4개를 세워 배치하고, 상단 헤드라인과 하단 핵심 포인트 3개.

**사진 배치 추천:** 상품 사진 없음. 실제 상품 사진이 생기면 히어로 중앙에 정면/사선 컷 배치.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 매일 쓰는 칫솔, 더 꼼꼼하게 │
│ 고르세요             │
├────────────────────┤
│     [칫솔 3~4개 히어로] │
├────────────────────┤
│ 칫솔모 | 헤드 | 그립감 │
└────────────────────┘
```

**상품 사실 정보:** 칫솔 카테고리만 확인됨.

**사용 사진:** 실제 칫솔 정면/사선 사진 필요.

**디자인 메모:** 흰색/민트/블루 계열의 청결한 느낌.

**최종 이미지 QA:** 브랜드명, 구성 수량, 기능성 표현을 임의로 만들지 않는다.

**확인 필요:** 브랜드명, 구성 수량, 색상, 칫솔모 강도.

## 컷 2. 문제 공감

**목적:** 아무 칫솔이나 고르면 생기는 불편을 보여준다.

**헤드라인:** 칫솔, 다 비슷해 보여도 사용감은 다릅니다

**서브카피:** 헤드 크기, 칫솔모 강도, 손잡이 그립감에 따라 매일의 양치감이 달라집니다.

**이미지 내 삽입 문구:** `헤드가 너무 큰가요?` / `칫솔모가 부담스럽나요?` / `손잡이가 미끄럽나요?`

**이미지 구성:** 세 가지 불편을 아이콘 카드로 보여주고 칫솔 실루엣 배치.

**사진 배치 추천:** 실제 제품의 헤드/손잡이 디테일 사진이 있으면 카드 옆에 배치.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 칫솔, 사용감은 다릅니다 │
├────────────────────┤
│ 큰 헤드? | 강한 모? | 미끄러운 그립? │
├────────────────────┤
│       [칫솔 실루엣]  │
└────────────────────┘
```

**상품 사실 정보:** 일반 구매 불안 기반.

**사용 사진:** 헤드, 칫솔모, 손잡이 사진.

**디자인 메모:** 문제 제기는 과장 없이 생활 불편 중심.

**최종 이미지 QA:** 치과 치료/질병 개선 암시 금지.

**확인 필요:** 제품의 실제 차별점.

## 컷 3. 해결 제안

**목적:** 이 칫솔이 어떤 기준으로 선택되는지 설명한다.

**헤드라인:** 세정감과 편안함을 함께 보는 칫솔

**서브카피:** 치아 사이를 닦는 칫솔모와 입 안에서 움직이기 쉬운 헤드 구조를 함께 확인하세요.

**이미지 내 삽입 문구:** `칫솔모 확인` / `헤드 크기 확인` / `그립감 확인`

**이미지 구성:** 칫솔을 분해하듯 헤드, 모, 손잡이 3영역 라벨링.

**사진 배치 추천:** 실제 제품 측면 사진에 라벨선을 연결.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 세정감과 편안함을 함께 │
├────────────────────┤
│ [헤드]──라벨          │
│ [칫솔모]──라벨        │
│ [손잡이]──라벨        │
└────────────────────┘
```

**상품 사실 정보:** 구조 설명은 일반 칫솔 기준.

**사용 사진:** 제품 전체 측면 사진.

**디자인 메모:** 인포그래픽 라벨은 짧고 크게.

**최종 이미지 QA:** 미확인 특허/기능 표현 금지.

**확인 필요:** 실제 헤드 크기, 칫솔모 소재/강도.

## 컷 4. 칫솔모 디테일

**목적:** 칫솔모에 대한 구매 불안을 줄인다.

**헤드라인:** 매일 닿는 칫솔모를 확인하세요

**서브카피:** 칫솔모의 굵기, 밀도, 끝 처리 방식은 실제 상품 정보 기준으로 표기해야 합니다.

**이미지 내 삽입 문구:** `칫솔모 디테일` / `강도 확인 필요` / `소재 확인 필요`

**이미지 구성:** 칫솔모 클로즈업과 정보 카드 3개.

**사진 배치 추천:** 실제 칫솔모 접사 사진 필수.

**사진 품질 판단:** 사진 없음. 접사 사진 확보 권장.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 매일 닿는 칫솔모 확인 │
├────────────────────┤
│   [칫솔모 접사 이미지] │
├────────────────────┤
│ 강도 | 소재 | 끝 처리 │
└────────────────────┘
```

**상품 사실 정보:** 미확인.

**사용 사진:** 칫솔모 접사.

**디자인 메모:** 칫솔모는 과장된 의료 효과 대신 구조 정보 중심.

**최종 이미지 QA:** 잇몸 질환 개선, 치석 제거 보장 표현 금지.

**확인 필요:** 부드러움 정도, 소재, 미세모 여부.

## 컷 5. 헤드 크기와 사용성

**목적:** 입 안에서 사용하기 편한지 설명한다.

**헤드라인:** 입 안 구석까지 움직이기 쉬운 헤드

**서브카피:** 실제 헤드 크기와 형태는 상품 사진과 스펙 기준으로 확인하세요.

**이미지 내 삽입 문구:** `헤드 크기` / `구석 사용감` / `실측 확인 필요`

**이미지 구성:** 칫솔 헤드 위주 이미지, 치아 아이콘, 실측 정보 자리.

**사진 배치 추천:** 실제 헤드 정면/측면 사진.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 움직이기 쉬운 헤드   │
├────────────────────┤
│ [칫솔 헤드 확대]     │
├────────────────────┤
│ 헤드 크기: 확인 필요 │
└────────────────────┘
```

**상품 사실 정보:** 미확인.

**사용 사진:** 헤드 확대 사진.

**디자인 메모:** 실측 숫자는 확인 후 입력.

**최종 이미지 QA:** 실제 스펙 없는 수치 금지.

**확인 필요:** 헤드 가로/세로 크기.

## 컷 6. 손잡이 그립감

**목적:** 사용 편의성을 보여준다.

**헤드라인:** 젖은 손에도 잡기 쉬운 그립감

**서브카피:** 손잡이 형태와 미끄럼 방지 구조는 실제 제품 사진 기준으로 안내합니다.

**이미지 내 삽입 문구:** `손잡이 형태` / `그립감 확인` / `소재 확인 필요`

**이미지 구성:** 손에 쥔 칫솔 장면과 손잡이 디테일 라벨.

**사진 배치 추천:** 손에 쥔 사용 장면 사진이 있으면 가장 좋음.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 잡기 쉬운 그립감     │
├────────────────────┤
│ [손에 쥔 칫솔 장면]  │
├────────────────────┤
│ 손잡이 | 소재 | 형태 │
└────────────────────┘
```

**상품 사실 정보:** 미확인.

**사용 사진:** 손잡이, 사용 장면.

**디자인 메모:** 생활용품답게 실사용 장면 중심.

**최종 이미지 QA:** 미끄럼 방지 보장 표현 금지.

**확인 필요:** 손잡이 소재, 표면 처리.

## 컷 7. 가족/개인 사용 구분

**목적:** 여러 개 구성 상품일 가능성에 맞춰 사용 상황을 제안한다.

**헤드라인:** 가족도, 1인가구도 구분해서 사용

**서브카피:** 색상과 구성 수량을 확인해 개인별로 위생적으로 나눠 사용하세요.

**이미지 내 삽입 문구:** `색상 구분` / `개인별 사용` / `구성 수량 확인 필요`

**이미지 구성:** 여러 색상의 칫솔을 나란히 배치하고 이름표 카드.

**사진 배치 추천:** 실제 색상 옵션 사진 필요.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 개인별로 구분해서 사용 │
├────────────────────┤
│ [색상별 칫솔 배열]    │
├────────────────────┤
│ 색상 | 구성 | 수량 확인 │
└────────────────────┘
```

**상품 사실 정보:** 색상/구성 미확인.

**사용 사진:** 색상 옵션 전체 사진.

**디자인 메모:** 구성 수량은 확정 전까지 비워둔다.

**최종 이미지 QA:** 실제 없는 색상/수량 노출 금지.

**확인 필요:** 색상 옵션, 세트 수량.

## 컷 8. 교체 주기 안내

**목적:** 재구매/세트 구매 이유를 만든다.

**헤드라인:** 칫솔은 주기적인 교체가 중요합니다

**서브카피:** 칫솔모가 벌어지거나 오염되기 전, 사용 상태를 보고 교체하세요.

**이미지 내 삽입 문구:** `교체 주기 체크` / `칫솔모 벌어짐 확인` / `위생적인 사용`

**이미지 구성:** 정상 칫솔모와 벌어진 칫솔모 비교 일러스트/사진.

**사진 배치 추천:** 실제 사용 전후 사진이 없으면 단순 인포그래픽으로 제작.

**사진 품질 판단:** 사진 없음, 인포그래픽 제작 권장.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 칫솔 교체 주기 체크 │
├─────────┬──────────┤
│ 정상 모  │ 벌어진 모 │
├─────────┴──────────┤
│ 상태를 보고 교체하세요 │
└────────────────────┘
```

**상품 사실 정보:** 일반 위생 안내.

**사용 사진:** 칫솔모 비교 이미지.

**디자인 메모:** 의료적 효과보다 위생적 사용 안내.

**최종 이미지 QA:** 특정 교체 기간은 근거 없으면 단정하지 않는다.

**확인 필요:** 제조사 권장 교체 주기.

## 컷 9. 구성·옵션 안내

**목적:** 구매 전 구성 정보를 정리한다.

**헤드라인:** 구매 전 구성 정보를 확인하세요

**서브카피:** 색상, 수량, 칫솔모 강도는 실제 옵션 기준으로 선택해야 합니다.

**이미지 내 삽입 문구:** `색상 확인 필요` / `수량 확인 필요` / `칫솔모 강도 확인 필요`

**이미지 구성:** 옵션 표와 칫솔 배열.

**사진 배치 추천:** 실제 패키지/구성품 전체 사진 필요.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 구성 정보를 확인하세요 │
├────────────────────┤
│ 색상 | 수량 | 모 강도 │
├────────────────────┤
│ [칫솔 구성 이미지]    │
└────────────────────┘
```

**상품 사실 정보:** 미확인.

**사용 사진:** 구성품 전체 사진.

**디자인 메모:** 옵션 표는 정확한 정보만 입력.

**최종 이미지 QA:** 수량/색상 허위 표기 금지.

**확인 필요:** 수량, 색상, 옵션명, 가격.

## 컷 10. 보관·사용 안내

**목적:** 구강용품의 위생적인 사용 방법을 안내한다.

**헤드라인:** 사용 후에는 깨끗하게 헹구고 건조하세요

**서브카피:** 물기를 털어 통풍되는 곳에 세워 보관하면 위생적으로 관리하기 좋습니다.

**이미지 내 삽입 문구:** `사용 후 헹굼` / `통풍 건조` / `개인별 보관`

**이미지 구성:** 칫솔 거치대, 물기 제거, 통풍 아이콘.

**사진 배치 추천:** 욕실/세면대 사용 장면.

**사진 품질 판단:** 사진 없음, 라이프스타일 이미지 제작 권장.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 깨끗하게 헹구고 건조 │
├────────────────────┤
│ 헹굼 | 건조 | 구분 보관 │
├────────────────────┤
│ [세면대 칫솔 보관 장면] │
└────────────────────┘
```

**상품 사실 정보:** 일반 보관 안내.

**사용 사진:** 욕실/세면대 컷.

**디자인 메모:** 습기/통풍을 상징하는 가벼운 아이콘.

**최종 이미지 QA:** 항균/살균 보장 표현 금지.

**확인 필요:** 제품별 보관 주의사항.

## 컷 11. FAQ·구매 불안 해소

**목적:** 구매 전 질문을 정리한다.

**헤드라인:** 구매 전 자주 묻는 질문

**서브카피:** 실제 옵션과 상품 정보를 확인하고 내 사용 습관에 맞게 선택하세요.

**이미지 내 삽입 문구:** `칫솔모 강도는?` / `구성 수량은?` / `색상은 랜덤인가요?` / `어린이도 사용 가능한가요?`

**이미지 구성:** FAQ 카드 4개.

**사진 배치 추천:** 패키지, 옵션, 칫솔모 사진이 있으면 작은 썸네일로 배치.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 구매 전 FAQ        │
├────────────────────┤
│ Q 칫솔모 강도는?    │
│ Q 구성 수량은?      │
│ Q 색상은 랜덤인가요? │
│ Q 어린이 사용 가능? │
└────────────────────┘
```

**상품 사실 정보:** 미확인.

**사용 사진:** 옵션/패키지 사진.

**디자인 메모:** 답변은 실제 상품 정보가 들어온 뒤 확정.

**최종 이미지 QA:** 연령 사용 가능 여부를 임의로 쓰지 않는다.

**확인 필요:** 연령 대상, 색상 랜덤 여부, 구성 수량.

## 컷 12. 최종 요약·CTA

**목적:** 핵심 구매 이유와 확인 항목을 정리한다.

**헤드라인:** 매일 쓰는 칫솔, 기준을 보고 선택하세요

**서브카피:** 칫솔모, 헤드, 그립감, 구성 정보를 확인하고 나에게 맞는 칫솔을 고르세요.

**이미지 내 삽입 문구:** `칫솔모 확인` / `헤드 확인` / `그립감 확인` / `구성 확인 후 구매`

**이미지 구성:** 칫솔 히어로 이미지와 4개 체크 카드.

**사진 배치 추천:** 실제 제품 대표 사진이 있으면 중앙 배치.

**사진 품질 판단:** 사진 없음, 판매용 초안.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 기준을 보고 선택하세요 │
├────────────────────┤
│ [대표 칫솔 이미지]    │
├────────────────────┤
│ 모 | 헤드 | 그립 | 구성 │
└────────────────────┘
```

**상품 사실 정보:** 칫솔 상품군 기준.

**사용 사진:** 대표 제품 사진.

**디자인 메모:** 최종 CTA는 차분하고 실용적으로.

**최종 이미지 QA:** 미확인 상품 스펙이 들어가지 않아야 한다.

**확인 필요:** 실제 상품명, 브랜드, 구성, 옵션.

## 4. 전체 디자인 톤앤매너

| 항목 | 권장 방향 |
|---|---|
| 무드 | 청결함, 실용성, 매일 쓰는 생활용품 |
| 컬러 | 화이트, 민트, 블루, 밝은 그레이 |
| 폰트 느낌 | 모바일에서 읽기 쉬운 굵은 고딕 |
| 이미지 스타일 | 쿠팡 실용 정보형, 칫솔 구조 라벨링, 옵션 카드 |
| 피해야 할 요소 | 치과 치료 효과, 항균/살균 보장, 인증/수치 조작 |

## 5. 준법·품질 체크

- `치주질환 예방`, `치석 제거 보장`, `잇몸 개선`, `항균 99.9%` 같은 표현은 근거 없으면 사용하지 않는다.
- 칫솔모 강도, 소재, 헤드 크기, 구성 수량, 색상은 실제 상품 정보 확인 후 확정한다.
- 어린이 사용 가능 여부, 식약처/인증, 항균 소재 여부는 자료 없으면 넣지 않는다.
- 사진이 없으므로 현재 기획은 `판매용 초안`이다.
