## Figma 전송 기준

Figma는 기본 선택이다.

적합한 경우:

- 디자인 시스템 관리
- 섹션별 컴포넌트화
- 웹 퍼블리싱 연결
- 반복 상세페이지 템플릿 제작

## Canva 전송 기준

Canva는 보조 선택이다.

적합한 경우:

- 빠른 마케팅 이미지 수정
- 팀원 공유
- 텍스트 교체 중심 작업
- SNS/배너 확장

## 완료 기준

- 모든 섹션 status가 approved
- 최종 상세페이지 초안 생성
- Figma 또는 Canva 전송 구조 생성
- 최종 JSON export_status 업데이트
