# Cojooboo 상세페이지 이미지 역기획

원본 URL: https://www.cojooboo.co.kr/courses/ed6f453f-be75-4f80-950d-e70ccf403ab1  
분석 기준: 저장된 상세 이미지 11개 + 대표 썸네일 1개  
용도: 그대로 복제하지 않고, 상세페이지 제작 스킬의 `이미지 기반 역기획` 템플릿으로 사용

## 한 줄 진단

이 상세페이지는 `고가 강의/코칭 상품`을 팔기 위한 모바일 랜딩형 상세페이지다. 핵심 설계는 기능 설명보다 `돈을 벌 수 있을 것 같은 기회`, `나도 할 수 있을 것 같은 낮은 리스크`, `강사의 실전성`, `마감/가격 압박`을 순서대로 누적하는 구조다.

## 설득 구조

1. `큰 약속`: 통장 건강, 노리스크, AI 건기식
2. `실체 제시`: 강의 일정과 커리큘럼
3. `실행 지원`: 챌린지, 피드백, 과제, 오프라인
4. `가격 압박`: 얼리버드, 마감일, 할인 전후 가격
5. `혜택 과다 제공`: VOD, 소싱, 자동화, 단톡방, 오프라인
6. `시장 욕망`: 건강, 젊음, 오래 살고 싶은 욕구
7. `AI 레버리지`: 시간은 줄이고 돈은 키우는 느낌
8. `리스크 제거`: 재고 없음, 제조 없음, 큰돈 필요 없음
9. `증거`: 수강생 매출, 입금 내역, 현장 사진
10. `강사 신뢰`: 자기소개, 전문성, 누적 경험
11. `마지막 확신`: 아직 기회가 많고 노하우를 알려주겠다는 약속

## 새 상세페이지 제작 시 추천 컷 구성

이 구조를 참고해 새 강의/코칭/서비스 상품을 만들 때는 아래 12컷 구성이 적합하다.

| 컷 | 목적 | 권장 카피 방향 | 이미지 구성 |
|---|---|---|---|
| 01 | 대표 오퍼 | `[결과 약속]을 [낮은 리스크/짧은 시간]으로` | 강사/상품 대표 이미지 + 강한 헤드라인 |
| 02 | 누구를 위한 상품인지 | `이런 분이라면 지금 필요합니다` | 체크리스트 3~5개 |
| 03 | 시장/문제 제시 | `이 문제는 앞으로 더 커집니다` | 시장 키워드, 욕망, 고통 포인트 |
| 04 | 새 해결 방식 | `기존 방식에 AI/시스템을 더하면 달라집니다` | Before/After 또는 공식 구조 |
| 05 | 리스크 제거 | `재고/제조/복잡한 세팅 없이 시작` | 장벽 제거 리스트 |
| 06 | 커리큘럼 | `몇 주차에 무엇을 완성하는지` | 회차별 카드형 구성 |
| 07 | 실행 프로그램 | `강의 후 혼자 남기지 않는 구조` | 챌린지, 과제, 피드백 구조 |
| 08 | 혜택 스택 | `구매자가 받는 모든 구성품` | 번호 리스트 10개 내외 |
| 09 | 증거 | `실제 사례/결과/후기` | 스크린샷, 현장 사진, 숫자 단 주의 |
| 10 | 강사 신뢰 | `왜 이 사람이 알려줄 수 있는지` | 프로필, 경력, 실전 장면 |
| 11 | 가격/마감 | `현재 가격과 마감 조건` | 할인 배지, 마감일, 가격표 |
| 12 | 마지막 CTA | `지금 신청해야 하는 이유` | 핵심 약속 재강조 + 버튼형 문구 |

## 이 스킬에 추가하면 좋은 기능

### URL 이미지 수집

- 사용자가 상세페이지 URL을 주면 Chrome 렌더링으로 DOM을 열고 이미지 URL을 추출한다.
- Next.js 이미지 프록시(`/_next/image`)는 원본 URL로 복원한다.
- Vercel 챌린지 때문에 단순 curl이 실패할 수 있으므로 브라우저 렌더링 경로를 우선한다.
- 다운로드 후 `images/`, `manifest.json`, `index.html`, `zip`을 자동 생성한다.

### 이미지 기반 역기획

- 컷 순서, 이미지 크기, 확장자, 반복 색상, 주요 레이아웃을 기록한다.
- 각 컷을 `오퍼`, `문제`, `시장논리`, `해결책`, `증거`, `가격`, `CTA` 같은 설득 기능으로 분류한다.
- 원본 문구를 그대로 베끼지 않고 `설득 구조`와 `카피 역할`만 추출한다.
- 새 상품에 적용할 10~12컷 제작 플랜으로 변환한다.

### 주의사항

- 경쟁사/타사 이미지는 참고와 구조 분석용으로만 사용한다.
- 최종 산출물은 원본 문구, 디자인, 강사 이미지, 수익 인증 이미지를 그대로 복제하지 않는다.
- 매출, 수익, 할인, 후기, 인증, 자격, 성과 숫자는 사용자가 제공하거나 검증된 것만 사용한다.
