## 2. 상세페이지 핵심 전략

사진 자료는 하우스자두의 원물 상태, 단면, 바구니 연출, 나무에 달린 모습, 크기 비교까지 갖추고 있어 상세페이지 기본 설득 흐름을 만들 수 있다. 다만 원산지, 품종, 중량, 당도, 선별 기준, 배송 방식, 보관법은 사진만으로 확정할 수 없으므로 이미지 안에서는 `확인 필요` 또는 일반 안내 문구로 처리한다.

구성은 첫 화면에서 제철감과 실물감을 보여주고, 중간 컷에서 과육·크기·농가 이미지를 통해 온라인 구매 불안을 낮춘 뒤, 마지막에는 구성/보관/배송 확인 항목을 정리하는 흐름이 적합하다.

## 4. 이미지 컷별 제작안

## 컷 1. 메인 히어로

**목적:** 첫 화면에서 하우스자두의 제철감과 실물감을 즉시 전달한다.

**헤드라인:** 한입에 퍼지는 제철 하우스자두

**서브카피:** 생생한 색감과 촉촉한 과육감을 사진으로 먼저 확인하세요.

**이미지 내 삽입 문구:** `하우스자두` / `제철 과일의 산뜻한 한입` / `실물 사진 기준 구성`

**이미지 구성:** 상단 큰 헤드라인, 중앙에 바구니 연출 자두, 하단에 핵심 포인트 3개.

**사진 배치 추천:** 이미지 1 또는 2의 바구니 구도를 참고하되 세로형 상세페이지용으로 더 깨끗한 히어로 이미지 재생성 권장.

**사진 품질 판단:** 재생성 권장. 원본은 구도는 좋지만 해상도와 세로 비율이 약하다.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 하우스자두          │
│ 제철 과일의 산뜻한 한입 │
├────────────────────┤
│                    │
│   [바구니 자두 히어로] │
│                    │
├────────────────────┤
│ 실물사진 | 과육감 | 크기확인 │
└────────────────────┘
```

**상품 사실 정보:** 사진상 자두 원물과 단면 확인 가능.

**사용 사진:** 이미지 1 또는 2 참고, 이미지 3 보조.

**디자인 메모:** 흰색/연한 크림 배경, 자두의 빨강·노랑·초록 색감을 살린다.

**최종 이미지 QA:** 상품명과 핵심 문구가 크게 보이고, 과장된 당도 표현이 없어야 한다.

**확인 필요:** 브랜드명, 산지, 중량, 판매 구성.

## 컷 2. 온라인 과일 구매 불안 공감

**목적:** 고객이 온라인에서 과일을 살 때 느끼는 불안을 먼저 짚는다.

**헤드라인:** 사진만 보고 사기 불안한 과일

**서브카피:** 크기, 익은 정도, 실제 색감까지 구매 전 확인할 수 있어야 합니다.

**이미지 내 삽입 문구:** `크기 확인` / `색감 확인` / `단면 확인` / `배송 상태 확인 필요`

**이미지 구성:** 좌측에는 질문형 카피, 우측에는 크기 비교와 단면 사진 콜라주.

**사진 배치 추천:** 이미지 11 크기 비교, 이미지 7 단면을 나란히 배치.

**사진 품질 판단:** 그대로 사용 가능. 다만 이미지 11의 작은 글자는 새로 크게 구성해야 한다.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 사진만 보고 사기 불안한 과일 │
├──────────┬─────────┤
│ [크기비교] │ [단면사진] │
├──────────┴─────────┤
│ 크기·색감·단면을 먼저 확인 │
└────────────────────┘
```

**상품 사실 정보:** 크기 비교 이미지와 단면 이미지 제공됨.

**사용 사진:** 이미지 11, 이미지 7.

**디자인 메모:** 체크리스트형 정보 카드로 구성한다.

**최종 이미지 QA:** 확인되지 않은 당도/등급을 단정하지 않는다.

**확인 필요:** 실제 판매 규격, 중량, 과수 크기 분류 기준.

## 컷 3. 실물 색감 소개

**목적:** 자두 색이 균일하지 않을 수 있음을 장점과 함께 설명한다.

**헤드라인:** 붉은빛과 노란빛이 함께 도는 실물 색감

**서브카피:** 하우스자두는 과실마다 색과 무늬가 조금씩 다를 수 있습니다.

**이미지 내 삽입 문구:** `실물 색감` / `개체별 색 차이 있음` / `사진 기준 안내`

**이미지 구성:** 자두 근접 사진을 배경으로 쓰고, 하단에 색 차이 안내 카드.

**사진 배치 추천:** 이미지 5 또는 6을 전체 배경으로 크롭.

**사진 품질 판단:** 보정 후 사용. 밝기와 색 균형을 조정하면 좋다.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 붉은빛과 노란빛의 실물 색감 │
├────────────────────┤
│ [자두 클로즈업 전체 배경] │
├────────────────────┤
│ 색상은 수확 시기와 개체별로 차이 │
└────────────────────┘
```

**상품 사실 정보:** 사진상 여러 색의 자두 확인 가능.

**사용 사진:** 이미지 5, 이미지 6.

**디자인 메모:** 자두 색을 과도하게 보정하지 않는다.

**최종 이미지 QA:** 실제 사진과 충돌하는 색감 보정 금지.

**확인 필요:** 선별 기준, 숙도 안내.

## 컷 4. 과육·단면 디테일

**목적:** 구매자가 가장 궁금해하는 속 과육과 먹는 느낌을 시각적으로 보여준다.

**헤드라인:** 단면으로 확인하는 촉촉한 과육

**서브카피:** 반으로 자른 모습과 조각 단면으로 과육감을 확인할 수 있습니다.

**이미지 내 삽입 문구:** `단면 확인` / `과육감` / `실물 촬영`

**이미지 구성:** 이미지 7을 중앙에 크게 넣고, 이미지 3의 단면 근접 컷을 보조로 배치.

**사진 배치 추천:** 이미지 7 직접 사용, 이미지 3은 우측 확대 원형 컷.

**사진 품질 판단:** 그대로 사용 가능.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 단면으로 확인하는 과육 │
├────────────────────┤
│ [통자두 / 반자두 / 조각] │
├────────────────────┤
│ 실물 촬영 기준 과육 안내 │
└────────────────────┘
```

**상품 사실 정보:** 단면 사진 있음.

**사용 사진:** 이미지 7, 이미지 3.

**디자인 메모:** 과육색이 잘 보이도록 배경은 밝게 유지.

**최종 이미지 QA:** 맛을 수치화하거나 당도를 단정하지 않는다.

**확인 필요:** 당도, 품종, 수확 시기.

## 컷 5. 크기 비교

**목적:** 온라인 과일 구매에서 중요한 크기 불안을 줄인다.

**헤드라인:** 눈으로 확인하는 하우스자두 크기

**서브카피:** 동전과 중소과·대과 비교 사진으로 실제 크기감을 가늠하세요.

**이미지 내 삽입 문구:** `크기 비교` / `중소과` / `대과` / `판매 규격 확인 필요`

**이미지 구성:** 이미지 11을 크게 사용하고, 텍스트를 다시 크게 디자인.

**사진 배치 추천:** 이미지 11 직접 사용하되 원본 라벨은 참고만 하고 새 정보 블록으로 재구성.

**사진 품질 판단:** 그대로 사용 가능.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 눈으로 확인하는 크기 │
├────────────────────┤
│ [동전] [중소과] [대과] │
├────────────────────┤
│ 실제 판매 규격은 옵션에서 확인 │
└────────────────────┘
```

**상품 사실 정보:** 동전, 중소과, 대과 비교 사진 있음.

**사용 사진:** 이미지 11.

**디자인 메모:** 라벨을 모바일에서 읽히도록 크게 재배치.

**최종 이미지 QA:** 판매 옵션이 확정되지 않았으면 `판매 규격 확인 필요` 표시.

**확인 필요:** 판매 옵션별 중량/과수.

## 컷 6. 농가·수확 이미지

**목적:** 산지와 수확 신뢰감을 준다.

**헤드라인:** 나무에서 익어가는 하우스자두

**이미지 내 삽입 문구:** `수확 이미지` / `농가 사진 제공` / `산지 정보 확인 필요`

**이미지 구성:** 이미지 10을 배경으로 사용하고, 좌측 상단에 짧은 스토리 카피.

**사진 배치 추천:** 이미지 10 직접 사용.

**사진 품질 판단:** 그대로 사용 가능.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 나무에서 익어가는 하우스자두 │
├────────────────────┤
│ [나무에 달린 자두 사진] │
├────────────────────┤
│ 산지/수확일은 확인 후 표기 │
└────────────────────┘
```

**상품 사실 정보:** 나무에 달린 자두 사진 제공.

**사용 사진:** 이미지 10.

**디자인 메모:** 초록 잎과 자두 색감을 자연스럽게 살린다.

**최종 이미지 QA:** 산지명을 임의로 넣지 않는다.

**확인 필요:** 산지, 농가명, 수확일.

## 컷 7. 선별·실물 상태

**목적:** 박스 안 실제 자두 상태와 풍성함을 보여준다.

**헤드라인:** 실제로 담기는 자두의 모습

## 컷 8. 맛 기대감

**사진 품질 판단:** 그대로 사용 가능. 더 고급스러운 식탁 컷은 재생성 가능.

## 컷 9. 구성·옵션 안내

**목적:** 구매 전 필요한 판매 단위 정보를 정리한다.

**헤드라인:** 구매 전 구성 정보를 확인하세요

**서브카피:** 중량, 과수, 크기 옵션은 실제 판매 옵션 기준으로 표기해야 합니다.

**이미지 내 삽입 문구:** `중량 확인 필요` / `과수 확인 필요` / `옵션 확인 필요`

**사진 배치 추천:** 이미지 5 또는 9를 배경으로 작게 사용.

**사진 품질 판단:** 보정 후 사용.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 구매 전 구성 정보 확인 │
├────────────────────┤
│ 중량 | 과수 | 크기 옵션 │
├────────────────────┤
│ [자두 원물 배경] │

```

**상품 사실 정보:** 크기 비교 사진 있음.

**사용 사진:** 이미지 5, 이미지 9, 이미지 11.

**디자인 메모:** 확인 전에는 구체 숫자를 넣지 않는다.

**확인 필요:** 판매 중량, 과수, 옵션명, 가격.

## 컷 10. 배송·수령 안내

**목적:** 신선식품 배송 불안을 줄인다.

**헤드라인:** 신선식품은 수령 후 확인이 중요합니다

**이미지 내 삽입 문구:** `수령 즉시 확인` / `무름 여부 확인` / `보관법 확인 필요`

**사진 품질 판단:** 보정 후 사용.

```text
┌────────────────────┐

│ 1 수령 즉시 확인   │
│ 2 보관 방법 확인   │
```

**상품 사실 정보:** 박스 안 자두 사진 제공.

**디자인 메모:** 정책 문구는 판매자 기준 확정 후 삽입.

**최종 이미지 QA:** 배송일/교환 보장을 임의로 넣지 않는다.

**확인 필요:** 배송 방식, 택배사, 보관법, 교환/환불 기준.

## 컷 11. FAQ·구매 불안 해소

**목적:** 색, 크기, 맛, 보관, 배송 관련 질문을 정리한다.

**헤드라인:** 구매 전 자주 묻는 질문

**서브카피:** 자연물 특성과 배송 조건을 미리 확인하면 더 안심하고 구매할 수 있습니다.

**이미지 내 삽입 문구:** `색이 다를 수 있나요?` / `크기는 어떻게 확인하나요?` / `보관은 어떻게 하나요?`

**사진 배치 추천:** 이미지 7, 이미지 11을 보조 썸네일로 사용.

**사진 품질 판단:** 그대로 사용 가능.

```text
┌────────────────────┐
```

**상품 사실 정보:** 사진상 색 차이, 크기 비교, 단면 확인 가능.

**최종 이미지 QA:** 미확인 보관법을 확정 문구로 쓰지 않는다.

**확인 필요:** 보관 권장 조건, 배송/교환 안내.

## 컷 12. 최종 요약·CTA

**목적:** 핵심 정보를 한 번 더 정리하고 구매 행동으로 연결한다.

**헤드라인:** 실물 사진으로 확인하고 고르는 하우스자두

**서브카피:** 색감, 단면, 크기 비교까지 보고 신중하게 선택하세요.

**이미지 내 삽입 문구:** `하우스자두` / `실물 사진 확인` / `크기 비교 확인` / `구성 정보 확인 후 구매`

**이미지 구성:** 히어로 이미지와 핵심 체크 4개를 함께 배치.

**사진 배치 추천:** 이미지 1 또는 2를 참고해 깔끔한 최종 CTA 컷 재생성.

**사진 품질 판단:** 재생성 권장. 최종 컷은 텍스트와 상품 배치가 정돈된 이미지가 필요하다.

**ASCII 레이아웃:**

```text
┌────────────────────┐
│ 실물 사진으로 확인하고 고르는 하우스자두 │
├────────────────────┤
│ [깔끔한 자두 히어로 이미지] │
├────────────────────┤
│ 색감 | 단면 | 크기 | 구성확인 │
└────────────────────┘
```

**상품 사실 정보:** 사진상 원물, 단면, 크기 비교 확인 가능.

**사용 사진:** 이미지 1/2 참고, 이미지 7/11 보조.

**디자인 메모:** 마지막 CTA는 과장 없이 정보 확인형으로 마무리.

**최종 이미지 QA:** `지금 바로 구매`보다 `구성 확인 후 구매`처럼 신선식품에 맞는 CTA 사용.

**확인 필요:** 판매자명, 구매 링크/채널, 최종 옵션.

## 5. 전체 디자인 톤앤매너

| 항목 | 권장 방향 |
|---|---|
| 무드 | 신선한 제철 과일, 농가 실물감, 과장 없는 신뢰형 |
| 컬러 | 자두 레드, 옐로우 그린, 잎사귀 그린, 흰색/크림 배경 |
| 폰트 느낌 | 모바일에서 잘 읽히는 굵은 고딕, 숫자/라벨은 큼직하게 |
| 이미지 스타일 | 원본 사진 중심, 히어로/CTA는 사진 참고 기반 재생성 권장 |
| 피해야 할 요소 | 과한 당도 보장, 산지/중량/리뷰 조작, 원본과 다른 색감 보정 |

## 6. 준법·품질 체크

- `최고 당도`, `무조건 달다`, `당도 보장`, `국내 최고` 같은 표현은 근거 없으면 사용하지 않는다.
- 산지, 품종, 중량, 과수, 크기 옵션, 수확일, 배송 방식, 보관법은 확인 후 확정 문구로 넣는다.
- 자연물 특성상 색, 크기, 모양 차이가 있을 수 있음을 안내한다.
- 배송 중 무름/파손/교환 기준은 판매자 정책 확인 후 별도 표기한다.
- 사진 속 자두 색감과 최종 생성 이미지 색감이 크게 달라지지 않게 한다.
