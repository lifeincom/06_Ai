# 판매처별 최종판매가 가격비교 Skill

## 목적
상품명과 상품코드를 입력받아 웹에서 동일 상품을 검색하고, 판매처별 최종판매가·배송비·판매 상태·검색 URL을 비교한다. 같은 상품코드가 여러 상품에 중복 입력되어도 각각 별도 상품으로 처리한다.

## 사용 상황
- 특정 상품의 현재 판매처별 가격을 비교할 때
- 상품명과 상품코드를 기준으로 최저가를 찾을 때
- 가격비교 사이트와 개별 쇼핑몰 가격이 다를 때 이를 구분해서 정리할 때
- 여러 상품을 한 번에 비교하되, 동일 상품코드가 중복될 수 있을 때

## 입력 형식
사용자는 아래 형식 중 하나로 입력한다.

### 단일 상품
```text
상품명: [비버리힐즈폴로클럽] 남여공용 스탠다드 폴로 반팔 카라 티셔츠
상품코드: TPX-001
```

### 복수 상품
```text
1. 상품명: [브랜드] 상품명 A
   상품코드: ABC-001
2. 상품명: [브랜드] 상품명 B
   상품코드: ABC-001
3. 상품명: [브랜드] 상품명 C
   상품코드: XYZ-002
```

### 표 형식
| 상품명 | 상품코드 |
|---|---|
| [브랜드] 상품명 A | ABC-001 |
| [브랜드] 상품명 B | ABC-001 |
| [브랜드] 상품명 C | XYZ-002 |

## 핵심 처리 원칙

### 1. 상품 식별 기준
- 상품코드만으로 중복 제거하지 않는다.
- 고유 상품 기준은 `상품명 + 상품코드 + 입력순번`이다.
- 동일 상품코드가 여러 번 입력되어도 각각 개별 검색 대상으로 유지한다.
- 검색 결과에서 상품코드가 일치하지 않으면 낮은 가격이어도 제외하거나 “품번 불일치 가능”으로 표시한다.

### 2. 검색 우선순위
각 상품마다 아래 순서로 검색한다.

1. `상품명 상품코드`
2. `"상품코드" "상품명 핵심 키워드"`
3. `상품코드 가격비교`
4. `상품명 최종판매가`
5. 주요 가격비교 사이트 검색 결과
6. 주요 쇼핑몰 개별 상품 페이지 검색 결과

### 3. 확인해야 할 사이트 유형
- 가격비교 사이트: 다나와, 에누리, 11번가 가격비교 등
- 오픈마켓: G마켓, 옥션, 11번가, 쿠팡 등
- 종합몰: SSG.COM, 롯데ON/롯데홈쇼핑, GS SHOP, CJ ONSTYLE, 현대Hmall, 홈&쇼핑 등
- 전문몰/공식몰: 브랜드 공식몰, 패션플러스, 하프클럽, 진마니아 등

### 4. 가격 산정 기준
표에 들어가는 가격은 아래 우선순위로 판단한다.

1. 상품 페이지 또는 가격비교 페이지에 표시된 최종판매가
2. 쿠폰 적용가 또는 혜택가가 명확히 표시된 경우 혜택가
3. 배송비가 명확하면 별도 열에 기재
4. 무료배송이면 `무료배송`으로 표시
5. 배송비가 불명확하면 `확인 필요`로 표시
6. 품절, 판매중지, 재입고알림 상품은 최저가 후보에서 제외하거나 상태 열에 명확히 표시

### 5. 경유 가격과 개별 페이지 가격 구분
가격비교 사이트의 경유 가격과 쇼핑몰 개별 상품페이지 가격이 다르면 다음처럼 구분한다.

- `옥션 / 다나와 경유`
- `G마켓 / 다나와 경유`
- `옥션 / 개별 상품페이지`
- `G마켓 / 개별 상품페이지`

같은 판매처라도 경유 조건이 다르면 별도 행으로 기록한다.

### 6. 중복 코드 처리
동일 상품코드가 여러 입력 상품에 반복되어도 삭제하지 않는다.

예시:
| 입력순번 | 상품명 | 상품코드 | 처리 |
|---:|---|---|---|
| 1 | 카라 티셔츠 | TPX-001 | 검색 진행 |
| 2 | 라운드 티셔츠 | TPX-001 | 검색 진행 |

최종 결과에는 입력순번을 포함하여 어떤 입력 상품의 검색 결과인지 구분한다.

## 출력 형식

### 1. 요약 결론
각 상품별로 1순위와 2순위 판매처를 먼저 제시한다.

```markdown
## 결론

### 상품 1: 상품명 / 상품코드
1순위: 판매처명 / 최종판매가 / 배송비
2순위: 판매처명 / 최종판매가 / 배송비
주의: 품번 혼재, 품절, 옵션가 변동 여부
```

### 2. 가격비교 표
```markdown
| 입력순번 | 상품명 | 상품코드 | 순위 | 판매처 | 최종판매가 | 배송비 | 상태 | 확인 기준 | URL |
|---:|---|---|---:|---|---:|---|---|---|---|
| 1 | 상품명 | TPX-001 | 1 | 옥션 / 다나와 경유 | 25,930원 | 무료배송 | 구매 가능 추정 | 가격비교 | URL |
```

### 3. 제외/주의 상품 표
품절, 품번 불일치, 세트상품, 옵션상품, 다른 품번은 별도 표로 분리한다.

```markdown
| 상품명 | 상품코드 | 판매처 | 표시가격 | 제외 이유 | URL |
|---|---|---|---:|---|---|
| 상품명 | TPX-001 | 쿠팡 | 24,000원 | 현재 판매 중 아님 | URL |
```

## 응답 규칙

- 사용자가 “다시 검색”이라고 하면 같은 방식으로 재검색한다.
- 사용자가 “엑셀로 저장”이라고 하면 판매처별 비교표와 URL을 포함해 엑셀 파일로 저장한다.
- 상품코드가 중복되어도 자동으로 합치지 않는다.
- 가격비교 결과는 날짜와 검색 기준을 함께 명시한다.
- 최저가라고 단정하기 전, 품절 여부와 상품코드 일치 여부를 확인한다.
- URL은 반드시 결과 표에 포함한다.
- 확인이 불가능한 항목은 추정하지 말고 `확인 필요`로 표시한다.

## 추천 검색 쿼리 템플릿

```text
"{상품코드}" "{상품명}"
"{상품코드}" "비버리힐즈폴로클럽"
"{상품명}" "{상품코드}" 가격비교
site:prod.danawa.com "{상품코드}"
site:catalog.11st.co.kr "{상품코드}"
site:gmarket.co.kr "{상품코드}"
site:auction.co.kr "{상품코드}"
site:coupang.com "{상품코드}"
site:ssg.com "{상품코드}"
site:gsshop.com "{상품코드}"
```

## 가격비교 판단 기준

### 구매 가능 최저가로 인정
- 상품코드 일치
- 상품명 핵심 키워드 일치
- 단품 상품
- 품절/판매중지 표시 없음
- 배송비 또는 무료배송 정보 확인 가능

### 주의 표시 필요
- 가격비교 사이트와 개별 쇼핑몰 가격이 다름
- 쿠폰 적용 여부가 불명확함
- 색상/사이즈 옵션 선택 시 가격 변동 가능
- 같은 품번이지만 세트/묶음 상품 가능성 있음

### 제외 후보
- 품절
- 재입고알림만 가능
- 판매중지
- 상품코드 불일치
- 2PACK/3PACK 등 세트상품인데 단품 가격처럼 보이는 경우
- 브랜드명만 같고 모델명이 다른 경우

## 최종 답변 예시

```markdown
창훈, 상품코드 TPX-001 기준으로 다시 확인하면 현재 최저가는 옥션 / 다나와 경유 25,930원 무료배송이야.

## 결론

### 1순위: 옥션 / 다나와 경유 25,930원 무료배송
현재 확인된 구매 가능 최저가야.

### 2순위: G마켓 / 다나와 경유 25,940원 무료배송
최저가와 10원 차이라 옥션 옵션 품절 시 대안으로 보면 돼.

## 판매처별 최종판매가 비교
| 순위 | 판매처 | 최종판매가 | 배송비 | 상태 | URL |
|---:|---|---:|---|---|---|
| 1 | 옥션 / 다나와 경유 | 25,930원 | 무료배송 | 구매 가능 추정 | URL |
| 2 | G마켓 / 다나와 경유 | 25,940원 | 무료배송 | 구매 가능 추정 | URL |
```
