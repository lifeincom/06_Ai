# Figma Export 구조

```text
Detail Page / [상품명]
├─ 00 Design System
├─ 01 Hero
├─ 02 Problem
├─ 03 Benefit
├─ 04 Usage
├─ 05 Detail
├─ 06 Spec
├─ 07 Guide
├─ 08 Closing
└─ 99 Final Preview
```

## 프레임 규격

| 프레임 | Width | Height |
|---|---:|---:|
| 각 섹션 | 860 | 1200 |
| 최종 프리뷰 | 860 | 전체 섹션 합산 |

## 작업 원칙

- 이미지 생성 결과는 섹션 배경/구도/무드 초안으로 사용
- 한글 텍스트는 Figma에서 편집 가능한 레이어로 재작성
- 디자인 시스템은 컴포넌트화
- 섹션별 프레임을 세로로 연결해 최종 상세페이지 구성
