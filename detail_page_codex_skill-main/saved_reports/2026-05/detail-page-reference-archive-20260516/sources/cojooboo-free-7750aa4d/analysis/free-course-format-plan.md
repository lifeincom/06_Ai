# Cojooboo 무료강의 상세페이지 포맷 역기획

원본 URL: https://www.cojooboo.co.kr/free-courses/7750aa4d-66b0-40f2-8a9d-e13135be1eb3  
상품명: `(6월 15일 19시 30분_하대표) 농산물 위탁판매로 온라인 월세 벌기`  
분석 기준: 대표 썸네일 1개 + 상세 이미지 2개  
포맷명: `무료강의 신청 전환형 / 리드마그넷형 상세페이지`

## 한 줄 진단

이 포맷은 유료 결제를 바로 받는 페이지가 아니라 `무료강의 신청`을 받기 위한 전환 페이지다. 그래서 가격 설득보다 `이 시장이 돈 된다`, `초보자도 가능하다`, `강사가 실제로 안다`, `무료라서 지금 신청해도 부담 없다`를 빠르게 누적한다.

## 유료 상세페이지형과의 차이

| 항목 | 유료 고가 강의형 | 무료강의 신청형 |
|---|---|---|
| 목표 | 바로 결제 | 무료 신청, DB 확보, 카톡/라이브 유입 |
| 이미지 수 | 10컷 이상으로 긴 설득 | 2장 안에 긴 섹션을 압축 |
| 가격 처리 | 할인 전후 가격, 마감, 혜택 스택 강조 | 무료, 일정, 신청 버튼 강조 |
| 신뢰 방식 | 커리큘럼, 혜택, 강사, 후기, 가격 방어 | 시장성, 성과 스크린샷, 강사 권위, 초보자 가능성 |
| CTA | 구매하기 | 무료강의 신청하기 |
| 핵심 감정 | 비싸지만 지금 사야 함 | 공짜니까 일단 들어봐야 함 |

## 대표 썸네일 구조

| 요소 | 역할 |
|---|---|
| 날짜/시간 배지 | 라이브 이벤트의 구체성 부여 |
| 큰 오퍼 | `농수산물 위탁판매로 온라인 월세 벌기` |
| 강사 얼굴 | 사람 기반 신뢰와 친근감 |
| 전문성 문구 | 쿠팡, 라이브커머스 전문가 하대표 |
| 어두운 배경 + 초록 포인트 | 돈/시장/농산물 카테고리의 시각 코드 |

## 상세 이미지 1 역할

상세 이미지 1은 `시장성과 가능성`을 설득하는 장이다.

| 구간 | 역할 | 메시지 |
|---|---|---|
| 오프닝 | 무료 보상 제시 | 신청만 해도 상담 전자책 2권, 무료 강의 일정 |
| 성과 증거 | 큰 숫자 제시 | 압도적 성과, 매출 사례 |
| 시장 정의 | 카테고리 소개 | 농수산물 위탁판매는 누구나 수익화 가능했던 시장 |
| 타깃 선별 | 이런 분들은 들어야 함 | 리스크 없이 온라인 판매를 원하는 사람 |
| 시장 기회 | 경쟁 시장 대비 | 일반 온라인 시장은 경쟁이 심하지만 농수산물은 기회가 있음 |
| 검색량/구매전환 | 수요 증거 | 검색량, 구매전환, 경쟁판이 비어 있음 |
| 리스크 제거 | 초보 진입 장벽 제거 | 재고, CS, 배송 부담을 낮춘 구조 |
| 강사 신뢰 | 전문가/수강생 인정 | 초대형 전문가, 인정하는 강사 |

## 상세 이미지 2 역할

상세 이미지 2는 `강사 신뢰와 신청 마감`을 설득하는 장이다.

| 구간 | 역할 | 메시지 |
|---|---|---|
| 숫자 증거 | 농수산물 매출/세금 사례 | 카테고리만으로 월매출, 부가세 면세 등 실전 사례 |
| 외부 인정 | 공공기관/강의 이력 | 지역 소상공인, 기관 강의, 현장 사진 |
| 강사 스토리 | 수익의 지름길 | 현장에서 직접 겪고 알려주는 노하우 |
| 상품 소싱 | 핵심 차별점 | 농수산물 판매의 가장 큰 벽은 상품 소싱 |
| 초보자 경고 | 싼 공급가 확보 어려움 | 혼자서는 좋은 가격에 공급받기 어렵다 |
| 해결책 제시 | 하대표가 대신 제공 | 전국 팔도의 좋은 물건, 잘 파는 방법 |
| 무료강의 혜택 | 배울 내용 요약 | 이커머스 A to Z, 성공의 레버리지 |
| 최종 CTA | 라이브 일정 재강조 | 6월 15일 19시 30분 무료 강의 |

## 설득 구조

1. `공짜 보상`: 신청만 해도 전자책/무료강의 제공
2. `큰 돈 냄새`: 매출 숫자와 성과 화면
3. `비어 있는 시장`: 농수산물 위탁판매는 아직 기회가 있다는 암시
4. `초보자 선별`: 리스크 없이 온라인 판매하고 싶은 사람 호출
5. `수요 증거`: 검색량, 구매전환, 경쟁 부족
6. `장벽 제거`: 재고, CS, 배송 부담을 낮춘다고 설득
7. `강사 권위`: 수강생 인정, 기관 인정, 현장 사진
8. `핵심 병목`: 결국 상품 소싱이 어렵다는 문제 제기
9. `해결 약속`: 좋은 물건과 판매 방법을 알려주겠다는 제안
10. `즉시 행동`: 무료 라이브 날짜와 시간 제시

## 이 포맷으로 새 상세페이지를 만들 때 추천 컷

무료강의/세미나/웨비나 신청 페이지는 2~4장짜리 긴 이미지 구성이 적합하다.

| 컷 | 목적 | 권장 구성 |
|---|---|---|
| 01 | 히어로 + 신청 명분 | 무료강의 날짜, 큰 결과 약속, 강사 얼굴, 무료 혜택 |
| 02 | 시장 기회 | 시장이 커지는 이유, 경쟁이 약한 지점, 수요 증거 |
| 03 | 초보자 리스크 제거 | 돈/시간/재고/기술 부담을 낮추는 설명 |
| 04 | 강사 신뢰 + CTA | 성과, 현장 사진, 기관/수강생 증거, 신청 마감 |

상세페이지 URL을 분석할 때 아래처럼 자동 분류하면 좋다.

| 분류 | 판별 기준 | 출력 방식 |
|---|---|---|
| `유료 고가 강의형` | 가격, 할인, 커리큘럼, 혜택 스택, 구매 버튼 | 10~12컷 기획 |
| `무료강의 신청형` | 무료, 라이브 일정, 신청 버튼, 마케팅 수신 동의 | 2~4장 긴 이미지 기획 |
| `전자책/자료 다운로드형` | 무료자료, PDF, 다운로드, 이메일/카톡 수집 | 1~3장 리드 수집 기획 |
| `상품 판매형` | 실제 물성 상품, 옵션, 배송, 성분/스펙 | 8~12컷 상품 상세 기획 |

## 제작 시 주의사항

- 원본의 강사 얼굴, 매출 인증, 카피, 색상 조합을 그대로 복제하지 않는다.
- `월매출`, `세금`, `수익`, `공급가`, `기관 인정` 같은 숫자와 신뢰 요소는 실제 증빙이 있을 때만 쓴다.
- 무료강의형은 최종 결제가 아니라 신청 전환이 목적이므로 CTA는 자주, 부담 없이, 날짜와 함께 반복한다.
- 긴 2장 구성은 모바일에서 스크롤 피로가 생길 수 있으므로 섹션 간 대비와 패턴 전환을 강하게 둔다.
