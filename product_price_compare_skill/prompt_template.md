# 가격비교 스킬 실행 프롬프트

아래 상품을 웹에서 검색하여 판매처별 최종판매가를 비교해줘.

## 검색 기준
- 상품명과 상품코드가 모두 일치하는 상품 우선
- 동일 상품코드가 중복되어도 상품명+입력순번 기준으로 각각 검색
- 품절/판매중지/재입고알림 상품은 최저가 후보에서 제외
- 가격비교 사이트 경유 가격과 쇼핑몰 개별 페이지 가격은 구분
- 최종판매가, 배송비, 상태, 확인 기준, URL 포함
- 최저가 1순위와 2순위 먼저 제시
- 가격 확인 날짜 명시

## 입력 상품
| 입력순번 | 상품명 | 상품코드 |

| 1 | [비버리힐즈폴로클럽] 남여공용 스탠다드 폴로 반팔 카라 티셔츠 | TPX-001 |
| 2 |  |  |
| 3 |  |  |

## 출력 형식
1. 상품별 결론
2. 판매처별 최종판매가 비교표
3. 제외/주의 상품 표
4. 구매 전 체크사항
