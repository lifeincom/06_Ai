# 07_ZIP 출력 프롬프트

모든 섹션 이미지 생성이 끝나면 아래 순서로 최종 결과물을 정리한다.

## 해야 할 일

1. 생성된 이미지 파일을 순서대로 확인한다.
2. 파일명을 정리한다.
3. 하나의 폴더에 모은다.
4. ZIP 파일로 압축한다.
5. 사용자에게 다운로드 링크를 제공한다.

## ZIP 파일명 규칙

```txt
상품명_detail_page_images.zip
```

## 최종 안내 형식

```md
# 최종 결과물

생성된 상세페이지 섹션 이미지입니다.

## 파일 목록

1. 01_hero.png
2. 02_problem.png
3. 03_benefit.png
4. 04_use_case.png
5. 05_detail.png
6. 06_comparison.png
7. 07_spec.png
8. 08_notice_cta.png

## 다운로드

[ZIP 다운로드](파일링크)
```
