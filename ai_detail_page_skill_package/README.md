# AI 상세페이지 제작 Skill 패키지

이 패키지는 ChatGPT에서 반복 상세페이지 제작 업무를 단계적으로 수행하기 위한 Skill 구조입니다.

## 목적

상품정보 입력부터 웹 리서치, 기획안, 디자인 시스템, 섹션별 이미지 프롬프트, JSON 관리, 수정, 최종 Figma/Canva 전송까지 하나의 워크플로우로 관리합니다.

## 핵심 파일

```text
SKILL.md
guides/
templates/
schemas/
examples/
outputs/
```

## 사용 방식

1. `SKILL.md`를 먼저 읽고 전체 역할과 실행 규칙을 확인합니다.
2. `guides/` 폴더의 단계별 지침서를 순서대로 실행합니다.
3. `schemas/`의 JSON 구조에 맞춰 프로젝트 정보를 관리합니다.
4. `templates/`의 프롬프트를 사용해 섹션별 이미지 생성과 수정을 진행합니다.
5. 최종 결과는 `outputs/` 폴더 구조에 맞춰 저장합니다.

## 권장 실행 순서

```text
01 상품정보 입력
02 웹 리서치
03 상세페이지 기획
04 디자인 시스템 설계
05 섹션별 이미지 프롬프트 작성
06 JSON 관리
07 수정 반영
08 최종 Export
```
