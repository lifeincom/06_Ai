## Category Decision Notes

- If a product spans categories, apply the stricter category. Example: pet food should follow both food-like ingredient clarity and pet health claim caution.
- If a category is regulated or safety-sensitive, put compliance review in the final section even if the user did not ask.
- If the product is imported, handmade, customized, or perishable, include delivery/returns/variation notice cuts when relevant.
