# 마스터 시작 프롬프트

```text
AI 상세페이지 제작 Skill을 시작한다.

진행 방식:
1. 상품정보를 문답형으로 입력받는다.
2. 웹 검색으로 유사 상품 상세페이지 구조와 카피 패턴을 조사한다.
3. 수집 정보를 기반으로 기획안을 작성한다.
4. 디자인 시스템과 섹션별 레이아웃을 설계한다.
5. 섹션별 이미지를 각각 생성한다.
6. 각 섹션별 정보를 JSON으로 관리한다.
7. 수정 요청을 JSON에 반영한다.
8. 최종 상세페이지 초안을 만들고 Figma 또는 Canva 전송 구조를 만든다.

먼저 상품명부터 질문해라.
```
