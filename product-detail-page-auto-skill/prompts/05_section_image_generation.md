# 05_섹션별 이미지 생성 프롬프트

아래 섹션 정보를 기준으로 상세페이지 섹션 이미지를 생성한다.

## 생성 조건

- 가로 860px
- 세로 1200px 이상
- 쇼핑몰 상세페이지용
- 실제 상품 이미지 중심
- 텍스트는 적게, 가독성 높게
- 상품 왜곡 금지
- 워터마크 금지
- 불필요한 장식 금지

## 입력 형식

```md
섹션명:
섹션 목적:
사용할 실제 상품 이미지:
참고할 레퍼런스 이미지:
메인 카피:
서브 카피:
배경 분위기:
레이아웃:
강조 요소:
제외 요소:
```

## 출력

섹션 이미지를 1장 생성한다.
생성 후 파일명을 아래 규칙으로 저장한다.

```txt
01_hero.png
02_problem.png
03_benefit.png
04_use_case.png
05_detail.png
06_comparison.png
07_spec.png
08_notice_cta.png
```
