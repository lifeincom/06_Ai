# 03_상세페이지 기획안 생성 프롬프트

아래 상품 정보를 바탕으로 쇼핑몰 상세페이지 기획안을 생성한다.

## 목표

- 상품의 핵심 USP를 상단 30% 안에 배치한다.
- 총 8개 섹션으로 구성한다.
- 각 섹션은 역할이 겹치지 않게 만든다.
- 실제 상품 이미지가 사용될 위치를 지정한다.
- 레퍼런스 이미지는 분위기와 구도 참고용으로만 사용한다.

## 출력 형식

```md
# 상세페이지 기획안

## 상품 핵심 전략
- 메인 USP:
- 구매 설득 포인트:
- 타깃 고객:
- 디자인 방향:

## 섹션 구성

### 01 HERO
- 목적:
- 핵심 카피:
- 서브 카피:
- 이미지 방향:
- 레이아웃:
- 포함 요소:
- 제외 요소:

### 02 PROBLEM
...

### 08 NOTICE / CTA
...
```
